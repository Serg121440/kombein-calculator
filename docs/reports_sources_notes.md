# Проект «Комбайн» — Заметка №2: Источники данных для отчётности

## Способы генерации отчётности и юнит-экономики

Система должна поддерживать **два способа** получения данных:

1. **Ручная загрузка отчётов** — пользователь выгружает файлы из личного кабинета маркетплейса и загружает в систему
1. **API** — автоматическое получение данных напрямую из маркетплейса

-----

## Ручная загрузка: список необходимых отчётов

### Отчёты о продажах

Дают информацию о количестве продаж, переходов, кликов, добавлений в корзину, заказов, отмен, рейтингов и прочих показателях воронки.

|Маркетплейс    |Название отчёта                          |
|---------------|-----------------------------------------|
|**Ozon**       |Отчёт по продажам (со всеми показателями)|
|**Wildberries**|Отчёт «Воронка продаж»                   |

### Отчёты по рекламным расходам

Дают информацию о затратах на продвижение в разрезе товаров и кампаний.

|Маркетплейс    |Название отчёта                |Детализация                   |
|---------------|-------------------------------|------------------------------|
|**Ozon**       |Отчёт по затратам на рекламу   |По товарам за выбранный период|
|**Wildberries**|Статистика расходования средств|По каждой рекламной кампании  |

### Отчёты для финансовой модели юнит-экономики

Необходимы дополнительно для расчёта по **Модели 2 (Финансовый результат)**.

|Маркетплейс    |Название отчёта |
|---------------|----------------|
|**Ozon**       |Отчёт по товарам|
|**Wildberries**|Финансовый отчёт|

-----

## Какие модели юнит-экономики закрывают отчёты

|Набор отчётов                                  |Доступные модели расчёта                              |
|-----------------------------------------------|------------------------------------------------------|
|Продажи + Реклама                              |Плановая (Модель 1) + По заказам с % выкупа (Модель 3)|
|Продажи + Реклама + Финотчёт / Отчёт по товарам|Финансовый результат (Модель 2)                       |
|API                                            |Все три модели (данные подтягиваются автоматически)   |

-----

## Итоговая схема

```
Источник данных
│
├── Ручная загрузка
│   ├── Отчёт продаж (Ozon) / Воронка продаж (WB)
│   ├── Реклама по товарам (Ozon) / Расходы по кампаниям (WB)
│   │       └──► Модели 1 и 3 (плановая + по заказам)
│   │
│   └── + Отчёт по товарам (Ozon) / Финотчёт (WB)
│               └──► Модель 2 (финансовый результат)
│
└── API
            └──► Все модели (автоматически)
```
