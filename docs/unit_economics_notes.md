# Проект «Комбайн» — Юнит-экономика

## Методологии расчёта

Юнит-экономика подразделяется на три модели расчёта.

-----

### Модель 1 — Плановая экономика

**Суть:** Прогнозная модель — что ожидаем получить.

**Входные данные:**

- Предполагаемый доход
- Расходы: комиссии, логистика
- Предположительный процент выкупа

**Результат:** Плановый P&L на единицу товара до начала продаж или при запуске новой позиции.

-----

### Модель 2 — По финансовому результату (фактический период)

**Суть:** Расчёт по итогам закрытого периода (например, месяц).

**Входные данные:**

- Фактическое количество продаж за период
- Фактическое количество выкупов
- Фактические списания: комиссии, логистика, рекламные расходы

**Механика распределения:**
Все затраты распределяются на **выкупленные** товары.

**Результат:** Модель близка к финансовому результату (P&L) — показывает реальную прибыль/убыток за период.

-----

### Модель 3 — Фактические затраты с учётом % выкупа

**Суть:** Промежуточная модель — затраты уже списаны, но выкуп ещё не завершён.

**Входные данные:**

- Фактически списанные затраты: комиссии, логистика, реклама
- Количество **проданных** (но ещё не выкупленных) товаров
- Процент выкупа и процент отмен

**Механика распределения:**
Фактические затраты распределяются на проданные товары с поправкой на ожидаемый процент выкупа и отмен.

**Результат:** Актуальная оценка юнит-экономики в моменте, без ожидания закрытия выкупов.

-----
