# Комбайн — юнит-экономика маркетплейсов

Веб-приложение для расчёта плановой и фактической юнит-экономики товаров на маркетплейсах **Ozon** и **Wildberries**.

## Стек

- Next.js 14 (App Router) + React 18 + TypeScript
- Tailwind CSS, Recharts, Zustand
- XLSX (SheetJS) — парсинг и экспорт Excel/CSV
- Хранение данных: `localStorage` (MVP, без серверной базы)

## Возможности MVP

- 🏬 **Магазины** — добавление магазинов Ozon/WB, маскирование API-ключей, лимиты по тарифам (Free/Pro/Enterprise)
- 📦 **Товары** — каталог с поиском, фильтрами, пагинацией, массовая плановая UE на единицу товара
- 📑 **Тарифы** — комиссия, эквайринг, логистика (с формулой `weight`/`volume`), хранение, последняя миля, периоды действия, импорт XLSX/CSV
- 📥 **Отчёты** — импорт финансовых отчётов Ozon/WB с автоопределением формата, прогресс, отчёт об ошибках, дедупликация по `orderId+date+amount+type+sku`
- 📊 **Дашборд** — KPI (выручка, прибыль, маржа, активные товары, отрицательная маржа), графики «План vs Факт», динамика, Топ-10 по прибыли, алерты
- 📈 **Аналитика** — сравнение текущего периода с предыдущим, метрики и подсветка трендов
- 📤 **Экспорт** — товары (CSV/XLSX) и транзакции (XLSX) с форматированием
- ⚙️ **Настройки** — дни хранения, тарифный план, сброс данных
- 🌱 **Демо-данные** — при первом запуске приложение наполняется примерами магазинов, товаров, тарифов и транзакций

## Запуск

```bash
npm install
npm run dev          # http://localhost:3000
npm run build        # продакшен-сборка
npm run typecheck    # проверка типов
```

## Структура

```
src/
  app/                # маршруты App Router
    page.tsx          # дашборд
    accounts/         # магазины
    products/         # товары + детальная карточка
    tariffs/          # тарифы (с импортом)
    reports/          # импорт отчётов
    analytics/        # тренды
    settings/         # настройки
  components/         # Sidebar, Topbar, Modal, KpiCard и т.д.
  lib/
    types.ts          # доменные типы
    store.ts          # zustand + persist
    economics.ts      # калькуляторы план/факт
    import.ts         # парсеры тарифов и отчётов
    export.ts         # CSV/XLSX экспорт
    seed.ts           # демо-данные
    format.ts         # форматирование чисел/дат
```

## Что осталось за рамками MVP (v2)

- Серверная авторизация (JWT, email-подтверждение, восстановление пароля)
- Реальное API-подключение к Ozon Seller / WB Suppliers
- Многопользовательский режим, шифрование API-ключей на сервере
- Модуль алертов с настраиваемыми порогами и историей
- Автосинхронизация отчётов
